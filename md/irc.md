# IRC (Internet Relay Chat)

## instal on linux

```bash
sudo apt update && sudo apt install irssi
```

## Launch irssi

```bash
irssi
```

## Connect to a server

```bash
/connect irc.libera.chat
```

## Setup a nickname

```bash
/nick <your-nickname>
```

## Register a nickname

```bash
/msg NickServ REGISTER <your-password> <youremail@example.com>
```
